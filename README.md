# epstein-files-island-visitors-and-name-frequency
Epstein Files: Island Visitors and Name Frequency Analysis

Our team use [Pardus AI](https://pardusai.org/) to get the report

## Description
This report analyzes individuals named in Department of Justice-released Epstein documents who have verified connections to Little St James island Epstein's private island in the U.S. Virgin Islands. The dataset includes 36 verified individuals with varying degrees of association - from confirmed island visitors to those with documented contact or flight manifest entries. Data was collected from flight manifests 1997-2006, court documents unsealed in January 2024, and verified news reports.
